# Security Policy

## Reporting a Vulnerability

We take the security of our systems seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

### How to Report

Please do **NOT** open an issue on GitHub for security vulnerabilities. Instead, please report it via email to:

**contact@gdgustp.com**

Please include as much information as possible to help us reproduce and validate the vulnerability.

### Response

We will acknowledge receipt of your report within 48 hours and will provide a timeline for addressing the vulnerability. We ask that you do not disclose the vulnerability publicly until we have had a chance to address it.
